Classpublic class ServiceEvent
InheritanceServiceEvent flash.events.Event

Events dispatched when interact with server. Depending on the request type, the data property type can be different. There is no individual events defined for each operation, reasons are: 1. simplified listening code. 2. similar convention to Google API's MapEvent with a feature property of multiple types.
